31) Meta Tag :
The <meta> HTML element represents metadata that cannot be represented by other meta-related elements, such as <base>, <link>, <script>, <style>, or <title>.

The type of metadata provided by the <meta> element can be one of the following:
i) If the name attribute is set, the <meta> element provides document-level metadata that applies to the whole page.
ii) If the http-equiv attribute is set, the <meta> element acts as a pragma directive to simulate directives that could otherwise be given by an HTTP header.
iii) If the charset attribute is set, the <meta> element is a charset declaration, giving the character encoding in which the document is encoded.
iV) If the itemprop attribute is set, the <meta> element provides user-defined metadata.

<meta name="viewport" content="width=device-width, initial-scale=1.0">
This is the important meta tag, Where it helps in the mobile devices.
